相对路径问题

相对路径参照物：命令行的工作目录

__dirname

__dirname 与 require 类似，都是 Node.js 环境中的’全局’变量 __dirname 保存着 当前文件所在目录的绝对路径 ，可以使用 __dirname 与文件名拼接成绝对路径

使用 fs 模块的时候，尽量使用 __dirname 将路径转化为绝对路径，这样可以避免相对路径产生的 Bug
